# Classic Snake
First project by using CodeX
A small dependency-free Python Snake game with deterministic core logic and a minimal Tkinter UI creating by CodeX.

## Run

```powershell
python -m snake_game.app
```

The game opens in a desktop window. Use arrow keys or WASD to move, `P` or the Pause button to pause, and `R`, `Space`, or the Restart button to restart after game over.

## Test

```powershell
python -m unittest discover
```

## Manual Check

- Move with arrow keys and WASD.
- Pause and resume with `P` or the Pause button.
- Eat food and confirm the snake grows and the score increases.
- Hit a wall or the snake body and confirm game over appears.
- Restart with the button, `R`, or `Space`.
